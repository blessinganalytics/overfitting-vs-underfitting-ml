ðŸ§  Key Concepts

Underfitting

Occurs when a model is too simple to capture patterns in the data, resulting in:

- High bias
- Poor training and testing performance

Good Fit

A balanced model that:

- Captures the underlying data pattern
- Generalizes well to unseen data

Overfitting

Happens when a model is too complex and memorizes training data instead of learning patterns, leading to:

- High variance
- Poor real-world prediction performance
